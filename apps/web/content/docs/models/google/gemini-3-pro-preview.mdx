# Gemini 3 Pro Preview

Google's latest flagship model with advanced reasoning capabilities.

## Specifications

| Property | Value |
|----------|-------|
| **Model ID** | `google/gemini-3-pro-preview` |
| **Context Window** | 1M tokens |
| **Input Price** | $0.002 / 1K tokens |
| **Output Price** | $0.012 / 1K tokens |
| **Capabilities** | text, tools, json, thinking |

## Quick Start

```typescript
import { generateText, createGateway } from 'ai';

const gateway = createGateway({
  apiKey: process.env.AI_GATEWAY_API_KEY
});

const { text } = await generateText({
  model: gateway('google/gemini-3-pro-preview'),
  prompt: 'Analyze the implications of quantum computing on current encryption standards',
});

console.log(text);
```

## Features

### Extended Thinking

Enable deep reasoning:

```typescript
const { text } = await generateText({
  model: gateway('google/gemini-3-pro-preview'),
  prompt: 'Develop a comprehensive strategy for AI safety',
  providerOptions: {
    google: {
      thinkingConfig: { thinkingBudget: 30000 },
    },
  },
});
```

### Complex Tool Chains

```typescript
import { jsonSchema } from 'ai';

const { toolCalls } = await generateText({
  model: gateway('google/gemini-3-pro-preview'),
  prompt: 'Research and analyze the competitive landscape for autonomous vehicles',
  tools: {
    webSearch: {
      description: 'Search the web',
      inputSchema: jsonSchema<{ query: string }>({
        type: 'object',
        properties: { query: { type: 'string' } },
        required: ['query'],
      }),
      execute: async ({ query }) => ({ results: ['Result 1', 'Result 2'] }),
    },
    analyzeCompany: {
      description: 'Analyze a company',
      inputSchema: jsonSchema<{ name: string }>({
        type: 'object',
        properties: { name: { type: 'string' } },
        required: ['name'],
      }),
      execute: async ({ name }) => ({ revenue: '$10B', marketShare: '15%' }),
    },
    generateReport: {
      description: 'Generate a structured report',
      inputSchema: jsonSchema<{ title: string; sections: string[] }>({
        type: 'object',
        properties: {
          title: { type: 'string' },
          sections: { type: 'array', items: { type: 'string' } },
        },
        required: ['title', 'sections'],
      }),
      execute: async (report) => report,
    },
  },
});
```

### Structured Output

```typescript
import { Output } from 'ai';
import { z } from 'zod/v4';

const { output } = await generateText({
  model: gateway('google/gemini-3-pro-preview'),
  prompt: 'Create a detailed technical architecture proposal',
  output: Output.object({
    schema: z.object({
      overview: z.string(),
      components: z.array(z.object({
        name: z.string(),
        purpose: z.string(),
        technologies: z.array(z.string()),
        interactions: z.array(z.string()),
      })),
      securityConsiderations: z.array(z.string()),
      scalabilityPlan: z.object({
        currentCapacity: z.string(),
        scalingStrategy: z.string(),
        bottlenecks: z.array(z.string()),
      }),
    }),
  }),
});
```
