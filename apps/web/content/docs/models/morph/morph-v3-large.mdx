# Morph V3 Large

The fastest code editing model available. **10,500 tokens per second** output speed.

## Specifications

| Property | Value |
|----------|-------|
| **Model ID** | `morph/morph-v3-large` |
| **Context Window** | 81K tokens |
| **Input Price** | $0.0009 / 1K tokens |
| **Output Price** | $0.0019 / 1K tokens |
| **Capabilities** | text, stream |
| **Speed** | 10,500 tokens/second |

## Quick Start

```typescript
import { generateText, createGateway } from 'ai';

const gateway = createGateway({
  apiKey: process.env.AI_GATEWAY_API_KEY
});

const { text } = await generateText({
  model: gateway('morph/morph-v3-large'),
  prompt: `Refactor this class to use composition over inheritance:

class Animal {
  constructor(name) {
    this.name = name;
  }
  speak() {
    console.log(this.name + ' makes a sound');
  }
}

class Dog extends Animal {
  speak() {
    console.log(this.name + ' barks');
  }
}`,
});

console.log(text);
```

## Features

### Ultra-Fast Generation

At 10,500 tokens/second, this is the fastest model for code output:

```typescript
import { streamText } from 'ai';

const startTime = Date.now();

const { textStream } = streamText({
  model: gateway('morph/morph-v3-large'),
  prompt: 'Generate a complete CRUD API in Express.js with TypeScript',
});

let tokens = 0;
for await (const chunk of textStream) {
  tokens += chunk.length;
  process.stdout.write(chunk);
}

console.log(`\n\nGenerated in ${Date.now() - startTime}ms`);
```

### Complex Refactoring

Handles larger, more complex code transformations:

```typescript
const { text } = await generateText({
  model: gateway('morph/morph-v3-large'),
  prompt: `Convert this React class component to a functional component with hooks:

class Counter extends React.Component {
  constructor(props) {
    super(props);
    this.state = { count: 0 };
  }

  componentDidMount() {
    document.title = \`Count: \${this.state.count}\`;
  }

  componentDidUpdate() {
    document.title = \`Count: \${this.state.count}\`;
  }

  increment = () => {
    this.setState({ count: this.state.count + 1 });
  };

  render() {
    return (
      <button onClick={this.increment}>
        Count: {this.state.count}
      </button>
    );
  }
}`,
});
```

## Best Use Cases

## Limitations

- **Code-focused**: Not for general chat or reasoning
- **No vision**: Cannot process images
- **No tool calling**: Cannot execute functions
- **No JSON output**: Returns plain text only
- **Smaller context**: 81K vs larger models
