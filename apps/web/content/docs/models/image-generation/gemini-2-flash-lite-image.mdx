# Gemini 2.5 Flash Lite Image

Cost-effective multimodal model for text and image generation.

## Specifications

| Property | Value |
|----------|-------|
| **Model ID** | `google/gemini-2.5-flash-lite-image` |
| **Context Window** | 1M tokens |
| **Pricing** | Per token (lowest tier) |
| **Type** | Multimodal (generateText) |

## Quick Start

```typescript
import { generateText, createGateway } from 'ai';

const gateway = createGateway({
  apiKey: process.env.AI_GATEWAY_API_KEY
});

const result = await generateText({
  model: gateway('google/gemini-2.5-flash-lite-image'),
  prompt: 'Create a simple icon for a music app',
  providerOptions: {
    google: {
      responseModalities: ['TEXT', 'IMAGE'],
    },
  },
});

console.log(result.text);
// result.files contains generated images
```

### Saving Images

```typescript
import fs from 'fs';

const result = await generateText({
  model: gateway('google/gemini-2.5-flash-lite-image'),
  prompt: 'Simple illustration of a coffee cup',
  providerOptions: {
    google: {
      responseModalities: ['TEXT', 'IMAGE'],
    },
  },
});

if (result.files) {
  for (const file of result.files) {
    if (file.uint8Array) {
      fs.writeFileSync('output.png', file.uint8Array);
    }
  }
}
```
