# FLUX Dev

Developer-focused model for testing and experimentation.

## Specifications

| Property | Value |
|----------|-------|
| **Model ID** | `bfl/flux-dev` |
| **Pricing** | Variable |
| **Type** | Image-only (generateImage) |

## Quick Start

```typescript
import { experimental_generateImage as generateImage } from 'ai';

const result = await generateImage({
  model: 'bfl/flux-dev',
  prompt: 'Test image for development',
  aspectRatio: '1:1',
});

const image = result.images[0];
console.log(image.base64);
```

## Features

### Development Testing

Ideal for integration testing:

```typescript
// Test your image generation pipeline
async function testImageGeneration() {
  const result = await generateImage({
    model: 'bfl/flux-dev',
    prompt: 'Simple test image',
    aspectRatio: '1:1',
  });

  console.assert(result.images.length > 0, 'Should generate an image');
  console.assert(result.images[0].base64, 'Should have base64 data');
}
```

### Rapid Iteration

Quick testing for prompt engineering:

```typescript
const prompts = [
  'Style A: modern minimalist',
  'Style B: vintage retro',
  'Style C: futuristic sci-fi',
];

for (const prompt of prompts) {
  const result = await generateImage({
    model: 'bfl/flux-dev',
    prompt,
    aspectRatio: '1:1',
  });
  // Evaluate results
}
```
