# GPT-5.1 Thinking

OpenAI's most capable reasoning model with explicit step-by-step thinking.

## Specifications

| Property | Value |
|----------|-------|
| **Model ID** | `openai/gpt-5.1-thinking` |
| **Context Window** | 512K tokens |
| **Input Price** | $0.0013 / 1K tokens |
| **Output Price** | $0.01 / 1K tokens |
| **Capabilities** | text, vision, tools, json, stream, thinking |

## Quick Start

```typescript
import { generateText, createGateway } from 'ai';

const gateway = createGateway({
  apiKey: process.env.AI_GATEWAY_API_KEY
});

const { text, reasoning } = await generateText({
  model: gateway('openai/gpt-5.1-thinking'),
  prompt: 'Solve: A train leaves Station A at 9 AM traveling 60 mph. Another train leaves Station B at 10 AM traveling 80 mph. If the stations are 280 miles apart, when do they meet?',
  providerOptions: {
    openai: {
      reasoningSummary: 'detailed',
    },
  },
});

console.log('Reasoning:', reasoning);
console.log('Answer:', text);
```

## Features

### Extended Thinking

Enable detailed reasoning process:

```typescript
const { text, reasoning } = await generateText({
  model: gateway('openai/gpt-5.1-thinking'),
  prompt: 'Analyze the competitive landscape for electric vehicles in 2026',
  providerOptions: {
    openai: {
      reasoningSummary: 'detailed', // or 'concise'
    },
  },
});

// reasoning contains the step-by-step thought process
// text contains the final conclusion
```

### Complex Problem Solving

```typescript
const { text, reasoning } = await generateText({
  model: gateway('openai/gpt-5.1-thinking'),
  prompt: `
    Debug this code and explain what's wrong:

    function fibonacci(n) {
      if (n <= 1) return 1;
      return fibonacci(n - 1) + fibonacci(n - 2);
    }

    console.log(fibonacci(5)); // Expected: 5, Got: 8
  `,
});
```

### Tool Calling with Reasoning

```typescript
import { jsonSchema } from 'ai';

const { toolCalls, reasoning } = await generateText({
  model: gateway('openai/gpt-5.1-thinking'),
  prompt: 'Find the best flight from NYC to London for under $500 next week',
  tools: {
    searchFlights: {
      description: 'Search for flights',
      inputSchema: jsonSchema<{ from: string; to: string; date: string; maxPrice: number }>({
        type: 'object',
        properties: {
          from: { type: 'string' },
          to: { type: 'string' },
          date: { type: 'string' },
          maxPrice: { type: 'number' },
        },
        required: ['from', 'to', 'date', 'maxPrice'],
      }),
      execute: async (params) => ({ flights: ['Flight 1', 'Flight 2'] }),
    },
  },
});
```

### Structured Output with Reasoning

```typescript
import { Output } from 'ai';
import { z } from 'zod/v4';

const { output, reasoning } = await generateText({
  model: gateway('openai/gpt-5.1-thinking'),
  prompt: 'Evaluate this business plan and provide a detailed analysis',
  output: Output.object({
    schema: z.object({
      viabilityScore: z.number().min(1).max(10),
      strengths: z.array(z.string()),
      weaknesses: z.array(z.string()),
      recommendations: z.array(z.object({
        action: z.string(),
        priority: z.enum(['high', 'medium', 'low']),
        rationale: z.string(),
      })),
      projectedSuccess: z.enum(['high', 'medium', 'low', 'uncertain']),
    }),
  }),
});

// reasoning shows HOW it arrived at these conclusions
```
