# Welcome to Layers

Layers is a platform for building human-AI coordination tools. It provides:

- **Unified Credit System** - Single balance across all AI operations
- **Multi-Provider Support** - 23 language models + 11 image models across 5 providers
- **Vercel AI Gateway** - Unified access to Anthropic, OpenAI, Google, Perplexity, Morph, and BFL

## Quick Start

```bash
# Install the SDK
bun add @layers/models @layers/credits

# Or with npm
npm install @layers/models @layers/credits
```
