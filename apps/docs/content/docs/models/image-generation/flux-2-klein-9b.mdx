# FLUX 2 Klein 9B

9B parameter model balancing quality and efficiency.

## Specifications

| Property | Value |
|----------|-------|
| **Model ID** | `bfl/flux-2-klein-9b` |
| **Parameters** | 9 billion |
| **Pricing** | $0.015 per megapixel |
| **Type** | Image-only (generateImage) |

## Quick Start

```typescript
import { experimental_generateImage as generateImage } from 'ai';

const result = await generateImage({
  model: 'bfl/flux-2-klein-9b',
  prompt: 'Professional product photography of a smartphone',
  aspectRatio: '1:1',
});

const image = result.images[0];
console.log(image.base64);
```
