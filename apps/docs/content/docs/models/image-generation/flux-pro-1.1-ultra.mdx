# FLUX 1.1 Pro Ultra

Enhanced quality version of FLUX 1.1 Pro.

## Specifications

| Property | Value |
|----------|-------|
| **Model ID** | `bfl/flux-pro-1.1-ultra` |
| **Pricing** | $0.06 per image |
| **Type** | Image-only (generateImage) |

## Quick Start

```typescript
import { experimental_generateImage as generateImage } from 'ai';

const result = await generateImage({
  model: 'bfl/flux-pro-1.1-ultra',
  prompt: 'Ultra-detailed fantasy landscape with dragon',
  aspectRatio: '16:9',
});

const image = result.images[0];
console.log(image.base64);
```

## Features

### Enhanced Details

Better fine details compared to standard 1.1 Pro:

```typescript
const result = await generateImage({
  model: 'bfl/flux-pro-1.1-ultra',
  prompt: 'Intricate mechanical watch with visible gears, macro photography',
  aspectRatio: '1:1',
});
```

### Fixed Pricing

Same per-image pricing model as 1.1 Pro:

```typescript
// All aspect ratios: $0.06 per image
const portrait = await generateImage({
  model: 'bfl/flux-pro-1.1-ultra',
  prompt: 'Detailed portrait painting',
  aspectRatio: '3:4',
});
```
