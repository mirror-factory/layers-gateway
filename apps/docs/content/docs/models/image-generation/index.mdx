## BFL Flux 2.x Models

The newest generation with megapixel-based pricing. Best quality and value.

```typescript
import { experimental_generateImage as generateImage } from 'ai';

// Flux 2.x - newest generation
const result = await generateImage({
  model: 'bfl/flux-2-pro',
  prompt: 'A serene mountain landscape at sunset',
  aspectRatio: '16:9',
});

const image = result.images[0];
console.log(image.base64);
```

## Gemini Multimodal

Text and image generation in a single call. These models can generate both text descriptions AND images.

```typescript
import { generateText, createGateway } from 'ai';

const gateway = createGateway({
  apiKey: process.env.AI_GATEWAY_API_KEY
});

const result = await generateText({
  model: gateway('google/gemini-3-pro-image'),
  prompt: 'Generate an image of a sunset over mountains and describe it',
  providerOptions: {
    google: {
      responseModalities: ['TEXT', 'IMAGE'],
    },
  },
});

// Text response
console.log(result.text);

// Images returned in result.files
if (result.files) {
  for (const file of result.files) {
    if (file.mediaType?.startsWith('image/')) {
      console.log('Got image:', file.mediaType, file.uint8Array?.length, 'bytes');
    }
  }
}
```
