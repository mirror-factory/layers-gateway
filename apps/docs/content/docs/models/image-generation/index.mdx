## BFL Flux Models

High-quality image generation with style control.

### Basic Generation

```typescript
import { experimental_generateImage as generateImage } from 'ai';

const result = await generateImage({
  model: 'bfl/flux-pro-1.1',
  prompt: 'A serene mountain landscape at sunset',
  aspectRatio: '16:9',
});

const image = result.images[0];
console.log(image.base64);
```

### Aspect Ratios

Supported: 1:1, 16:9, 9:16, 4:3, 3:4, 21:9

## Gemini Multimodal

Text and image generation in a single call.

```typescript
import { generateText, createGateway } from 'ai';

const gateway = createGateway({
  apiKey: process.env.AI_GATEWAY_API_KEY
});

const result = await generateText({
  model: gateway('google/gemini-3-pro-image'),
  prompt: 'Generate an image of a sunset over mountains',
  providerOptions: {
    google: {
      responseModalities: ['TEXT', 'IMAGE'],
    },
  },
});

// Images returned in result.files
if (result.files) {
  for (const file of result.files) {
    if (file.mediaType?.startsWith('image/')) {
      console.log('Got image:', file.mediaType);
    }
  }
}
```
