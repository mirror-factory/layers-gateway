# FLUX 1.1 Pro

BFL's previous generation professional model with per-image pricing.

## Specifications

| Property | Value |
|----------|-------|
| **Model ID** | `bfl/flux-pro-1.1` |
| **Pricing** | $0.04 per image |
| **Type** | Image-only (generateImage) |

## Quick Start

```typescript
import { experimental_generateImage as generateImage } from 'ai';

const result = await generateImage({
  model: 'bfl/flux-pro-1.1',
  prompt: 'A beautiful sunset over the ocean',
  aspectRatio: '16:9',
});

const image = result.images[0];
console.log(image.base64);
```

## Features

### Simple Pricing

Fixed per-image cost regardless of resolution:

```typescript
// All these cost $0.04 each
const images = await Promise.all([
  generateImage({ model: 'bfl/flux-pro-1.1', prompt: 'Square image', aspectRatio: '1:1' }),
  generateImage({ model: 'bfl/flux-pro-1.1', prompt: 'Wide image', aspectRatio: '16:9' }),
  generateImage({ model: 'bfl/flux-pro-1.1', prompt: 'Tall image', aspectRatio: '9:16' }),
]);
```

### Professional Quality

Suitable for professional applications:

```typescript
const result = await generateImage({
  model: 'bfl/flux-pro-1.1',
  prompt: 'Corporate headshot, professional lighting, neutral background',
  aspectRatio: '3:4',
});
```

## Best Use Cases

- When predictable per-image cost is preferred
