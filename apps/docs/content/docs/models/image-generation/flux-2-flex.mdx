# FLUX 2 Flex

Versatile image model for varied creative styles.

## Specifications

| Property | Value |
|----------|-------|
| **Model ID** | `bfl/flux-2-flex` |
| **Pricing** | $0.06 per megapixel |
| **Type** | Image-only (generateImage) |

## Quick Start

```typescript
import { experimental_generateImage as generateImage } from 'ai';

const result = await generateImage({
  model: 'bfl/flux-2-flex',
  prompt: 'Watercolor painting of a serene garden',
  aspectRatio: '4:3',
});

const image = result.images[0];
console.log(image.base64);
```

## Features

### Style Flexibility

FLUX 2 Flex excels at diverse artistic styles:

```typescript
// Illustration style
const illustration = await generateImage({
  model: 'bfl/flux-2-flex',
  prompt: 'Children book illustration of a friendly dragon',
  aspectRatio: '4:3',
});

// Digital art style
const digitalArt = await generateImage({
  model: 'bfl/flux-2-flex',
  prompt: 'Cyberpunk city with neon lights, digital art',
  aspectRatio: '16:9',
});

// Oil painting style
const painting = await generateImage({
  model: 'bfl/flux-2-flex',
  prompt: 'Oil painting of a still life with fruit and flowers',
  aspectRatio: '1:1',
});
```

### All Aspect Ratios

```typescript
const ratios = ['1:1', '16:9', '9:16', '4:3', '3:4', '21:9'];

for (const ratio of ratios) {
  const result = await generateImage({
    model: 'bfl/flux-2-flex',
    prompt: 'Beautiful landscape scene',
    aspectRatio: ratio,
  });
  // Process each image
}
```
