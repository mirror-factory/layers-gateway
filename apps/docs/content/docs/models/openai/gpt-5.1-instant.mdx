# GPT-5.1 Instant

Optimized for the lowest possible latency while maintaining high quality.

## Specifications

| Property | Value |
|----------|-------|
| **Model ID** | `openai/gpt-5.1-instant` |
| **Context Window** | 512K tokens |
| **Input Price** | $0.0013 / 1K tokens |
| **Output Price** | $0.01 / 1K tokens |
| **Capabilities** | text, vision, tools, json, stream |

## Quick Start

```typescript
import { generateText, createGateway } from 'ai';

const gateway = createGateway({
  apiKey: process.env.AI_GATEWAY_API_KEY
});

const { text } = await generateText({
  model: gateway('openai/gpt-5.1-instant'),
  prompt: 'Quick question: what is the capital of France?',
});

console.log(text); // Fastest response time
```

## Features

### Real-Time Chat

Perfect for interactive applications:

```typescript
import { streamText } from 'ai';

const { textStream } = streamText({
  model: gateway('openai/gpt-5.1-instant'),
  prompt: 'Help me brainstorm ideas for a mobile app',
});

// Ultra-fast time to first token
for await (const chunk of textStream) {
  process.stdout.write(chunk);
}
```

### Vision (Image Input)

```typescript
const { text } = await generateText({
  model: gateway('openai/gpt-5.1-instant'),
  messages: [{
    role: 'user',
    content: [
      { type: 'text', text: 'What is this?' },
      { type: 'image', image: imageBuffer },
    ],
  }],
});
```

### Tool Calling

```typescript
import { jsonSchema } from 'ai';

const { toolCalls } = await generateText({
  model: gateway('openai/gpt-5.1-instant'),
  prompt: 'What time is it?',
  tools: {
    getTime: {
      description: 'Get current time',
      inputSchema: jsonSchema<{ timezone?: string }>({
        type: 'object',
        properties: { timezone: { type: 'string' } },
      }),
      execute: async ({ timezone }) => new Date().toLocaleString(),
    },
  },
});
```

### Structured Output

```typescript
import { Output } from 'ai';
import { z } from 'zod/v4';

const { output } = await generateText({
  model: gateway('openai/gpt-5.1-instant'),
  prompt: 'Parse: John, 25, NYC',
  output: Output.object({
    schema: z.object({
      name: z.string(),
      age: z.number(),
      city: z.string(),
    }),
  }),
});
```
