# GPT-5 Chat

OpenAI's latest generation chat model with improved reasoning and conversational abilities.

## Specifications

| Property | Value |
|----------|-------|
| **Model ID** | `openai/gpt-5-chat` |
| **Context Window** | 512K tokens |
| **Input Price** | $0.0013 / 1K tokens |
| **Output Price** | $0.01 / 1K tokens |
| **Capabilities** | text, vision, tools, json, stream |

## Quick Start

```typescript
import { generateText, createGateway } from 'ai';

const gateway = createGateway({
  apiKey: process.env.AI_GATEWAY_API_KEY
});

const { text } = await generateText({
  model: gateway('openai/gpt-5-chat'),
  prompt: 'Help me understand quantum entanglement',
});

console.log(text);
```

## Features

### Extended Context (512K)

Handle much longer conversations and documents:

```typescript
const { text } = await generateText({
  model: gateway('openai/gpt-5-chat'),
  messages: [
    { role: 'user', content: veryLongDocument }, // Up to 512K tokens
    { role: 'user', content: 'Summarize the main themes' },
  ],
});
```

### Vision (Image Input)

```typescript
const { text } = await generateText({
  model: gateway('openai/gpt-5-chat'),
  messages: [{
    role: 'user',
    content: [
      { type: 'text', text: 'Analyze this chart and explain the trends' },
      { type: 'image', image: chartImage },
    ],
  }],
});
```

### Tool Calling

```typescript
import { jsonSchema } from 'ai';

const { toolCalls } = await generateText({
  model: gateway('openai/gpt-5-chat'),
  prompt: 'Schedule a meeting for next Tuesday at 2pm',
  tools: {
    createEvent: {
      description: 'Create a calendar event',
      inputSchema: jsonSchema<{ title: string; date: string; time: string }>({
        type: 'object',
        properties: {
          title: { type: 'string' },
          date: { type: 'string' },
          time: { type: 'string' },
        },
        required: ['title', 'date', 'time'],
      }),
      execute: async (event) => ({ created: true, id: '12345' }),
    },
  },
});
```

### Structured Output

```typescript
import { Output } from 'ai';
import { z } from 'zod/v4';

const { output } = await generateText({
  model: gateway('openai/gpt-5-chat'),
  prompt: 'Analyze this customer feedback and categorize the issues',
  output: Output.object({
    schema: z.object({
      overallSentiment: z.enum(['positive', 'neutral', 'negative']),
      categories: z.array(z.object({
        category: z.string(),
        count: z.number(),
        examples: z.array(z.string()),
      })),
      recommendations: z.array(z.string()),
    }),
  }),
});
```
