## Quick Start

```typescript
import { generateText, createGateway } from 'ai';

const gateway = createGateway({ apiKey: process.env.AI_GATEWAY_API_KEY });

const { text } = await generateText({
  model: gateway('google/gemini-2.5-flash'),
  prompt: 'Explain the solar system',
});
```

## Key Features

### 1M Token Context

Gemini models support up to 1 million tokens of context:

```typescript
const { text } = await generateText({
  model: gateway('google/gemini-2.5-pro'),
  messages: [
    { role: 'user', content: veryLongDocument }, // Up to 1M tokens
    { role: 'user', content: 'Summarize the key points' },
  ],
});
```

### Image Generation (Flash Image & Pro Image)

```typescript
const result = await generateText({
  model: gateway('google/gemini-3-pro-image'),
  prompt: 'Generate an image of a sunset over mountains',
  providerOptions: {
    google: {
      responseModalities: ['TEXT', 'IMAGE'],
    },
  },
});

// Images returned in result.files
for (const file of result.files || []) {
  if (file.mediaType?.startsWith('image/')) {
    fs.writeFileSync('output.png', file.uint8Array);
  }
}
```

### Extended Thinking (Pro models)

```typescript
const { text } = await generateText({
  model: gateway('google/gemini-2.5-pro'),
  prompt: 'Complex reasoning task...',
  providerOptions: {
    google: {
      thinkingConfig: { thinkingBudget: 10000 },
    },
  },
});
```

### Vision (Image Input)

```typescript
const { text } = await generateText({
  model: gateway('google/gemini-2.5-flash-lite'),
  messages: [{
    role: 'user',
    content: [
      { type: 'text', text: 'What objects are in this image?' },
      { type: 'image', image: imageBuffer },
    ],
  }],
});
```
