# Gemini 2.5 Flash

Fast, versatile model with the largest context window available.

## Specifications

| Property | Value |
|----------|-------|
| **Model ID** | `google/gemini-2.5-flash` |
| **Context Window** | 1M tokens |
| **Input Price** | $0.0003 / 1K tokens |
| **Output Price** | $0.0025 / 1K tokens |
| **Capabilities** | text, vision, tools, json |

## Quick Start

```typescript
import { generateText, createGateway } from 'ai';

const gateway = createGateway({
  apiKey: process.env.AI_GATEWAY_API_KEY
});

const { text } = await generateText({
  model: gateway('google/gemini-2.5-flash'),
  prompt: 'Explain the differences between SQL and NoSQL databases',
});

console.log(text);
```

## Features

### 1M Token Context

Process extremely long documents:

```typescript
const { text } = await generateText({
  model: gateway('google/gemini-2.5-flash'),
  messages: [
    { role: 'user', content: entireBookContent }, // Up to 1 million tokens
    { role: 'user', content: 'Summarize the main themes and character arcs' },
  ],
});
```

### Vision (Image Input)

```typescript
const { text } = await generateText({
  model: gateway('google/gemini-2.5-flash'),
  messages: [{
    role: 'user',
    content: [
      { type: 'text', text: 'Describe what you see in this image' },
      { type: 'image', image: imageBuffer },
    ],
  }],
});
```

### Tool Calling

```typescript
import { jsonSchema } from 'ai';

const { toolCalls } = await generateText({
  model: gateway('google/gemini-2.5-flash'),
  prompt: 'What is the weather in Tokyo?',
  tools: {
    getWeather: {
      description: 'Get weather for a city',
      inputSchema: jsonSchema<{ city: string }>({
        type: 'object',
        properties: { city: { type: 'string' } },
        required: ['city'],
      }),
      execute: async ({ city }) => ({ temp: 22, conditions: 'cloudy' }),
    },
  },
});
```

### Structured Output

```typescript
import { Output } from 'ai';
import { z } from 'zod/v4';

const { output } = await generateText({
  model: gateway('google/gemini-2.5-flash'),
  prompt: 'Generate a recipe for chocolate cake',
  output: Output.object({
    schema: z.object({
      name: z.string(),
      prepTime: z.string(),
      cookTime: z.string(),
      ingredients: z.array(z.object({
        item: z.string(),
        amount: z.string(),
      })),
      instructions: z.array(z.string()),
    }),
  }),
});
```
