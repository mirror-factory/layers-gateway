# Perplexity Models

Sonar series models with built-in web search for real-time information retrieval.

## Models

| Model | Context | Input $/1K | Output $/1K | Capabilities |
|-------|---------|------------|-------------|--------------|
| [Sonar](/docs/models/perplexity/sonar) | 127K | $0.001 | $0.001 | web, json, stream |
| [Sonar Pro](/docs/models/perplexity/sonar-pro) | 200K | $0.003 | $0.015 | web, json, stream |
| [Sonar Reasoning Pro](/docs/models/perplexity/sonar-reasoning-pro) | 127K | $0.002 | $0.008 | web, thinking |

## Quick Start

```typescript
import { generateText, createGateway } from 'ai';

const gateway = createGateway({ apiKey: process.env.AI_GATEWAY_API_KEY });

// Sonar automatically searches the web
const { text } = await generateText({
  model: gateway('perplexity/sonar-pro'),
  prompt: 'What are the latest developments in AI as of today?',
});
// Response includes web-sourced information with citations
```

## Key Features

### Built-in Web Search

Sonar models automatically search the web for current information:

```typescript
const { text } = await generateText({
  model: gateway('perplexity/sonar'),
  prompt: 'What is the current stock price of Apple?',
});
```

### Structured Output with Web Data

```typescript
import { Output } from 'ai';
import { z } from 'zod/v4';

const { output } = await generateText({
  model: gateway('perplexity/sonar'),
  prompt: 'Find the top 3 programming languages in 2026',
  output: Output.object({
    schema: z.object({
      languages: z.array(z.object({
        name: z.string(),
        rank: z.number(),
        reason: z.string(),
      })),
    }),
  }),
});
```

### Reasoning with Web Search

```typescript
const { text } = await generateText({
  model: gateway('perplexity/sonar-reasoning-pro'),
  prompt: 'Research and analyze: What factors contributed to the latest tech layoffs?',
});
// Combines multi-step reasoning with web search
```

## Important Notes

1. **No simple text prompts**: Sonar models are optimized for web search queries. Simple prompts like "Say hello" will return clarification questions.

2. **Citations included**: Responses include source citations from web search.

3. **Best use cases**:
   - Current events queries
   - Research questions
   - Fact-checking
   - Questions requiring up-to-date information
