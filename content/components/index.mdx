Interactive UI components from `@layers/ui`. Each component is displayed using Storybook with live examples.

## Installation

```bash
npm install @layers/ui
# or
bun add @layers/ui
```

## Using Components

```tsx
import { Button, Card, Input } from '@layers/ui';

function MyComponent() {
  return (
    <Card>
      <Input placeholder="Enter text..." />
      <Button variant="primary">Submit</Button>
    </Card>
  );
}
```

## Testing Components

Components are tested using:

1. **Storybook Interaction Tests** - User flow simulation
2. **Visual Regression** - Screenshot comparison
3. **Accessibility** - a11y addon checks

See [MFDR-010: Testing Strategy](/docs/mfdr-010) for details.
