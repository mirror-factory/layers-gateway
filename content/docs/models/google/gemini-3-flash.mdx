# Gemini 3 Flash

The newest generation of Google's fast multimodal model.

## Specifications

| Property | Value |
|----------|-------|
| **Model ID** | `google/gemini-3-flash` |
| **Context Window** | 1M tokens |
| **Input Price** | $0.0005 / 1K tokens |
| **Output Price** | $0.003 / 1K tokens |
| **Capabilities** | vision, tools, json |

## Quick Start

```typescript
import { generateText, createGateway } from 'ai';

const gateway = createGateway({
  apiKey: process.env.AI_GATEWAY_API_KEY
});

const { text } = await generateText({
  model: gateway('google/gemini-3-flash'),
  prompt: 'Explain containerization in simple terms',
});

console.log(text);
```

## Features

### Vision (Image Input)

```typescript
const { text } = await generateText({
  model: gateway('google/gemini-3-flash'),
  messages: [{
    role: 'user',
    content: [
      { type: 'text', text: 'Analyze this diagram and explain the workflow' },
      { type: 'image', image: diagramImage },
    ],
  }],
});
```

### Tool Calling

```typescript
import { jsonSchema } from 'ai';

const { toolCalls } = await generateText({
  model: gateway('google/gemini-3-flash'),
  prompt: 'Book a flight from NYC to LA',
  tools: {
    searchFlights: {
      description: 'Search for available flights',
      inputSchema: jsonSchema<{ from: string; to: string; date: string }>({
        type: 'object',
        properties: {
          from: { type: 'string' },
          to: { type: 'string' },
          date: { type: 'string' },
        },
        required: ['from', 'to', 'date'],
      }),
      execute: async (params) => ({ flights: ['Flight 1', 'Flight 2'] }),
    },
  },
});
```

### Structured Output

```typescript
import { Output } from 'ai';
import { z } from 'zod/v4';

const { output } = await generateText({
  model: gateway('google/gemini-3-flash'),
  prompt: 'Generate a product listing',
  output: Output.object({
    schema: z.object({
      name: z.string(),
      description: z.string(),
      price: z.number(),
      category: z.string(),
      tags: z.array(z.string()),
    }),
  }),
});
```
