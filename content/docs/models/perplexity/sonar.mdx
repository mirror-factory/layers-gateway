# Perplexity Sonar

Fast, affordable model with built-in web search for real-time information retrieval.

## Specifications

| Property | Value |
|----------|-------|
| **Model ID** | `perplexity/sonar` |
| **Context Window** | 127K tokens |
| **Input Price** | $0.001 / 1K tokens |
| **Output Price** | $0.001 / 1K tokens |
| **Capabilities** | web, json, stream |

## Quick Start

```typescript
import { generateText, createGateway } from 'ai';

const gateway = createGateway({
  apiKey: process.env.AI_GATEWAY_API_KEY
});

const { text } = await generateText({
  model: gateway('perplexity/sonar'),
  prompt: 'What are the latest developments in AI?',
});

console.log(text);
// Response includes web-sourced information with citations
```

## Features

### Automatic Web Search

Sonar automatically searches the web for current information. No configuration needed.

```typescript
const { text } = await generateText({
  model: gateway('perplexity/sonar'),
  prompt: 'What is the current stock price of Apple?',
});
```

### Structured Output

Extract structured data from web searches:

```typescript
import { Output } from 'ai';
import { z } from 'zod/v4';

const { output } = await generateText({
  model: gateway('perplexity/sonar'),
  prompt: 'Find the top 3 JavaScript frameworks in 2026',
  output: Output.object({
    schema: z.object({
      frameworks: z.array(z.object({
        name: z.string(),
        description: z.string(),
        popularity: z.string(),
      })),
    }),
  }),
});

console.log(output.frameworks);
```

### Streaming

```typescript
import { streamText } from 'ai';

const { textStream } = streamText({
  model: gateway('perplexity/sonar'),
  prompt: 'Summarize today\'s tech news',
});

for await (const chunk of textStream) {
  process.stdout.write(chunk);
}
```

## Best Use Cases

- Current events queries
- Real-time information lookup
- Fact-checking with sources
- Quick research questions
- Stock prices and market data

## Limitations

- **Not for simple prompts**: Optimized for web search queries. Simple prompts like "Say hello" may return clarification questions.
- **No tool calling**: Cannot execute custom functions
- **No vision**: Cannot process images
