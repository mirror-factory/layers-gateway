# Claude 4.5 Opus

The most capable Claude model, optimized for complex reasoning and the highest quality outputs.

## Specifications

| Property | Value |
|----------|-------|
| **Model ID** | `anthropic/claude-opus-4.5` |
| **Context Window** | 200K tokens |
| **Input Price** | $0.005 / 1K tokens |
| **Output Price** | $0.025 / 1K tokens |
| **Capabilities** | text, vision, tools, json, stream, thinking |

## Quick Start

```typescript
import { generateText, createGateway } from 'ai';

const gateway = createGateway({
  apiKey: process.env.AI_GATEWAY_API_KEY
});

const { text } = await generateText({
  model: gateway('anthropic/claude-opus-4.5'),
  prompt: 'Design a distributed system architecture for a real-time multiplayer game',
});

console.log(text);
```

## Features

### Extended Thinking

Opus excels at complex reasoning with extended thinking:

```typescript
const { text, reasoning } = await generateText({
  model: gateway('anthropic/claude-opus-4.5'),
  prompt: 'Analyze this research paper and identify potential flaws in the methodology...',
  providerOptions: {
    anthropic: {
      thinking: { type: 'enabled', budgetTokens: 20000 },
    },
  },
});

console.log('Reasoning process:', reasoning);
console.log('Analysis:', text);
```

### Complex Tool Chains

```typescript
import { jsonSchema } from 'ai';

const { toolCalls } = await generateText({
  model: gateway('anthropic/claude-opus-4.5'),
  prompt: 'Research the company ACME Corp and generate a financial analysis report',
  tools: {
    searchWeb: {
      description: 'Search the web for information',
      inputSchema: jsonSchema<{ query: string }>({
        type: 'object',
        properties: { query: { type: 'string' } },
        required: ['query'],
      }),
      execute: async ({ query }) => ({ results: ['Financial data...'] }),
    },
    getStockData: {
      description: 'Get stock market data',
      inputSchema: jsonSchema<{ symbol: string }>({
        type: 'object',
        properties: { symbol: { type: 'string' } },
        required: ['symbol'],
      }),
      execute: async ({ symbol }) => ({ price: 150.25, change: 2.5 }),
    },
    generateReport: {
      description: 'Generate a formatted report',
      inputSchema: jsonSchema<{ title: string; sections: string[] }>({
        type: 'object',
        properties: {
          title: { type: 'string' },
          sections: { type: 'array', items: { type: 'string' } },
        },
        required: ['title', 'sections'],
      }),
      execute: async ({ title, sections }) => ({ report: `# ${title}\n${sections.join('\n')}` }),
    },
  },
});
```

### Structured Output

```typescript
import { Output } from 'ai';
import { z } from 'zod/v4';

const { output } = await generateText({
  model: gateway('anthropic/claude-opus-4.5'),
  prompt: 'Create a comprehensive project plan for building a mobile app',
  output: Output.object({
    schema: z.object({
      projectName: z.string(),
      phases: z.array(z.object({
        name: z.string(),
        duration: z.string(),
        tasks: z.array(z.object({
          name: z.string(),
          assignee: z.string(),
          priority: z.enum(['high', 'medium', 'low']),
        })),
        deliverables: z.array(z.string()),
      })),
      risks: z.array(z.object({
        description: z.string(),
        mitigation: z.string(),
        likelihood: z.enum(['high', 'medium', 'low']),
      })),
      timeline: z.string(),
    }),
  }),
});
```
