# Claude 4.5 Sonnet

The best balance of speed, quality, and cost. Ideal for most use cases including coding and complex chat.

## Specifications

| Property | Value |
|----------|-------|
| **Model ID** | `anthropic/claude-sonnet-4.5` |
| **Context Window** | 200K tokens |
| **Input Price** | $0.003 / 1K tokens |
| **Output Price** | $0.015 / 1K tokens |
| **Capabilities** | text, vision, tools, json, stream, thinking |

## Quick Start

```typescript
import { generateText, createGateway } from 'ai';

const gateway = createGateway({
  apiKey: process.env.AI_GATEWAY_API_KEY
});

const { text } = await generateText({
  model: gateway('anthropic/claude-sonnet-4.5'),
  prompt: 'Write a TypeScript function to validate email addresses',
});

console.log(text);
```

## Features

### Extended Thinking

Enable step-by-step reasoning for complex problems:

```typescript
const { text, reasoning } = await generateText({
  model: gateway('anthropic/claude-sonnet-4.5'),
  prompt: 'Solve this logic puzzle: ...',
  providerOptions: {
    anthropic: {
      thinking: { type: 'enabled', budgetTokens: 10000 },
    },
  },
});

console.log('Reasoning:', reasoning);
console.log('Answer:', text);
```

### Vision (Image Input)

```typescript
const { text } = await generateText({
  model: gateway('anthropic/claude-sonnet-4.5'),
  messages: [{
    role: 'user',
    content: [
      { type: 'text', text: 'Review this UI mockup and suggest improvements' },
      { type: 'image', image: mockupImage },
    ],
  }],
});
```

### Tool Calling

```typescript
import { jsonSchema } from 'ai';

const { toolCalls } = await generateText({
  model: gateway('anthropic/claude-sonnet-4.5'),
  prompt: 'Search for the latest news about AI',
  tools: {
    webSearch: {
      description: 'Search the web',
      inputSchema: jsonSchema<{ query: string }>({
        type: 'object',
        properties: { query: { type: 'string' } },
        required: ['query'],
      }),
      execute: async ({ query }) => ({ results: ['AI news result 1', 'AI news result 2'] }),
    },
  },
});
```

### Structured Output

```typescript
import { Output } from 'ai';
import { z } from 'zod/v4';

const { output } = await generateText({
  model: gateway('anthropic/claude-sonnet-4.5'),
  prompt: 'Generate a product specification for a smart home device',
  output: Output.object({
    schema: z.object({
      name: z.string(),
      features: z.array(z.string()),
      price: z.number(),
      specifications: z.object({
        dimensions: z.string(),
        weight: z.string(),
        connectivity: z.array(z.string()),
      }),
    }),
  }),
});
```

### Streaming

```typescript
import { streamText } from 'ai';

const { textStream } = streamText({
  model: gateway('anthropic/claude-sonnet-4.5'),
  prompt: 'Write a short story about a robot learning to paint',
});

for await (const chunk of textStream) {
  process.stdout.write(chunk);
}
```
