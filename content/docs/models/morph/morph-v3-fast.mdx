# Morph V3 Fast

Fast code editing model optimized for quick transformations and edits.

## Specifications

| Property | Value |
|----------|-------|
| **Model ID** | `morph/morph-v3-fast` |
| **Context Window** | 81K tokens |
| **Input Price** | $0.0008 / 1K tokens |
| **Output Price** | $0.0012 / 1K tokens |
| **Capabilities** | text, stream |

## Quick Start

```typescript
import { generateText, createGateway } from 'ai';

const gateway = createGateway({
  apiKey: process.env.AI_GATEWAY_API_KEY
});

const { text } = await generateText({
  model: gateway('morph/morph-v3-fast'),
  prompt: `Refactor this function to use async/await:

function fetchData() {
  return fetch('/api/data')
    .then(res => res.json())
    .then(data => data.items);
}`,
});

console.log(text);
```

## Features

### Code Refactoring

Transform code patterns quickly:

```typescript
const { text } = await generateText({
  model: gateway('morph/morph-v3-fast'),
  prompt: `Convert this JavaScript to TypeScript:

function add(a, b) {
  return a + b;
}

function greet(name) {
  return 'Hello, ' + name;
}`,
});
```

### Apply Diffs

Apply code changes efficiently:

```typescript
const { text } = await generateText({
  model: gateway('morph/morph-v3-fast'),
  prompt: `Apply this change to the code:

Original:
const config = {
  debug: false,
  timeout: 5000,
};

Change: Add a 'retries' property set to 3

Return the complete updated code.`,
});
```

### Streaming Output

Stream generated code for real-time display:

```typescript
import { streamText } from 'ai';

const { textStream } = streamText({
  model: gateway('morph/morph-v3-fast'),
  prompt: 'Generate a React component for a toggle button',
});

for await (const chunk of textStream) {
  process.stdout.write(chunk);
}
```

## Best Use Cases

- Quick code refactoring
- Syntax transformations
- Adding/removing code patterns
- Style conversions (e.g., callbacks to async/await)
- Simple code generation

## Limitations

- **Code-focused**: Not for general chat or reasoning
- **No vision**: Cannot process images
- **No tool calling**: Cannot execute functions
- **No JSON output**: Returns plain text only
- **Smaller context**: 81K vs larger models
