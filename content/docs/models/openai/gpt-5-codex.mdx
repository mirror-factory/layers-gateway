# GPT-5 Codex

Specialized model optimized for code generation, analysis, and programming tasks.

## Specifications

| Property | Value |
|----------|-------|
| **Model ID** | `openai/gpt-5-codex` |
| **Context Window** | 512K tokens |
| **Input Price** | $0.0013 / 1K tokens |
| **Output Price** | $0.01 / 1K tokens |
| **Capabilities** | tools, json, stream |

## Quick Start

Codex models are optimized for tool use, not plain text prompts:

```typescript
import { generateText, createGateway, jsonSchema } from 'ai';

const gateway = createGateway({
  apiKey: process.env.AI_GATEWAY_API_KEY
});

const { toolCalls } = await generateText({
  model: gateway('openai/gpt-5-codex'),
  prompt: 'Create a React component for a login form',
  tools: {
    writeCode: {
      description: 'Write code to a file',
      inputSchema: jsonSchema<{ filename: string; code: string }>({
        type: 'object',
        properties: {
          filename: { type: 'string' },
          code: { type: 'string' },
        },
        required: ['filename', 'code'],
      }),
      execute: async ({ filename, code }) => ({ written: filename }),
    },
  },
});
```

## Features

### Code Generation via Tools

```typescript
const { toolCalls } = await generateText({
  model: gateway('openai/gpt-5-codex'),
  prompt: 'Build a REST API with Express.js for a todo app',
  tools: {
    writeFile: {
      description: 'Write a file',
      inputSchema: jsonSchema<{ path: string; content: string }>({
        type: 'object',
        properties: {
          path: { type: 'string' },
          content: { type: 'string' },
        },
        required: ['path', 'content'],
      }),
      execute: async ({ path, content }) => ({ created: path }),
    },
    runCommand: {
      description: 'Run a shell command',
      inputSchema: jsonSchema<{ command: string }>({
        type: 'object',
        properties: { command: { type: 'string' } },
        required: ['command'],
      }),
      execute: async ({ command }) => ({ output: 'Success' }),
    },
  },
});
```

### Code Review

```typescript
const { toolCalls } = await generateText({
  model: gateway('openai/gpt-5-codex'),
  prompt: 'Review this code for bugs and security issues:\n' + codeToReview,
  tools: {
    reportIssue: {
      description: 'Report a code issue',
      inputSchema: jsonSchema<{ line: number; severity: string; message: string }>({
        type: 'object',
        properties: {
          line: { type: 'number' },
          severity: { type: 'string' },
          message: { type: 'string' },
        },
        required: ['line', 'severity', 'message'],
      }),
      execute: async (issue) => issue,
    },
  },
});
```

### Structured Output for Code

```typescript
import { Output } from 'ai';
import { z } from 'zod/v4';

const { output } = await generateText({
  model: gateway('openai/gpt-5-codex'),
  prompt: 'Analyze this codebase and identify the architecture pattern',
  output: Output.object({
    schema: z.object({
      pattern: z.string(),
      components: z.array(z.object({
        name: z.string(),
        type: z.string(),
        responsibilities: z.array(z.string()),
      })),
      dependencies: z.array(z.object({
        from: z.string(),
        to: z.string(),
        type: z.string(),
      })),
    }),
  }),
});
```

## Important Notes

1. **Use tools, not plain text**: Codex models are optimized for tool/function calling
2. **No vision support**: Cannot process images
3. **Large context**: 512K tokens for analyzing large codebases
