# GPT-4o Mini

The most cost-effective OpenAI model, ideal for high-volume applications and simple tasks.

## Specifications

| Property | Value |
|----------|-------|
| **Model ID** | `openai/gpt-4o-mini` |
| **Context Window** | 128K tokens |
| **Input Price** | $0.0001 / 1K tokens |
| **Output Price** | $0.0006 / 1K tokens |
| **Capabilities** | text, vision, tools, json, stream |

## Quick Start

```typescript
import { generateText, createGateway } from 'ai';

const gateway = createGateway({
  apiKey: process.env.AI_GATEWAY_API_KEY
});

const { text } = await generateText({
  model: gateway('openai/gpt-4o-mini'),
  prompt: 'Summarize this article in 3 bullet points',
});

console.log(text);
```

## Features

### Vision (Image Input)

```typescript
const { text } = await generateText({
  model: gateway('openai/gpt-4o-mini'),
  messages: [{
    role: 'user',
    content: [
      { type: 'text', text: 'What text is shown in this image?' },
      { type: 'image', image: imageBuffer },
    ],
  }],
});
```

### Tool Calling

```typescript
import { jsonSchema } from 'ai';

const { toolCalls } = await generateText({
  model: gateway('openai/gpt-4o-mini'),
  prompt: 'Convert 100 USD to EUR',
  tools: {
    convertCurrency: {
      description: 'Convert between currencies',
      inputSchema: jsonSchema<{ amount: number; from: string; to: string }>({
        type: 'object',
        properties: {
          amount: { type: 'number' },
          from: { type: 'string' },
          to: { type: 'string' },
        },
        required: ['amount', 'from', 'to'],
      }),
      execute: async ({ amount, from, to }) => ({ result: amount * 0.92 }),
    },
  },
});
```

### Structured Output

```typescript
import { Output } from 'ai';
import { z } from 'zod/v4';

const { output } = await generateText({
  model: gateway('openai/gpt-4o-mini'),
  prompt: 'Extract contact info: John Smith, john@example.com, 555-1234',
  output: Output.object({
    schema: z.object({
      name: z.string(),
      email: z.string(),
      phone: z.string(),
    }),
  }),
});
```

### Streaming

```typescript
import { streamText } from 'ai';

const { textStream } = streamText({
  model: gateway('openai/gpt-4o-mini'),
  prompt: 'Generate a short poem about coding',
});

for await (const chunk of textStream) {
  process.stdout.write(chunk);
}
```
