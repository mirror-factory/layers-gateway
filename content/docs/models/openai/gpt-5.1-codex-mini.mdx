# GPT-5.1 Codex Mini

Cost-effective code generation for high-volume applications.

## Specifications

| Property | Value |
|----------|-------|
| **Model ID** | `openai/gpt-5.1-codex-mini` |
| **Context Window** | 512K tokens |
| **Input Price** | $0.0003 / 1K tokens |
| **Output Price** | $0.002 / 1K tokens |
| **Capabilities** | tools, json, stream |

## Quick Start

```typescript
import { generateText, createGateway, jsonSchema } from 'ai';

const gateway = createGateway({
  apiKey: process.env.AI_GATEWAY_API_KEY
});

const { toolCalls } = await generateText({
  model: gateway('openai/gpt-5.1-codex-mini'),
  prompt: 'Write a utility function to validate email addresses',
  tools: {
    writeCode: {
      description: 'Write code',
      inputSchema: jsonSchema<{ filename: string; code: string }>({
        type: 'object',
        properties: {
          filename: { type: 'string' },
          code: { type: 'string' },
        },
        required: ['filename', 'code'],
      }),
      execute: async ({ filename, code }) => ({ written: filename }),
    },
  },
});
```

### Batch Processing

Ideal for processing many code files:

```typescript
async function processFiles(files: string[]) {
  const results = await Promise.all(files.map(async (file) => {
    const { toolCalls } = await generateText({
      model: gateway('openai/gpt-5.1-codex-mini'),
      prompt: `Add JSDoc comments to this file:\n${file}`,
      tools: {
        documentCode: {
          description: 'Add documentation',
          inputSchema: jsonSchema<{ code: string }>({
            type: 'object',
            properties: { code: { type: 'string' } },
            required: ['code'],
          }),
          execute: async ({ code }) => code,
        },
      },
    });
    return toolCalls;
  }));
  return results;
}
```
