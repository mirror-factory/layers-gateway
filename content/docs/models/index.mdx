## Quick Start

Use any model with the Layers API:

```bash
curl -X POST https://layers.hustletogether.com/api/v1/chat/completions \
  -H "Authorization: Bearer lyr_live_YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"model": "anthropic/claude-sonnet-4-20250514", "messages": [{"role": "user", "content": "Hello!"}]}'
```
