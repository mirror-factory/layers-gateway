# Imagen 4 Ultra

Google's premium image generation with maximum quality.

## Specifications

| Property | Value |
|----------|-------|
| **Model ID** | `google/imagen-4.0-ultra-generate-001` |
| **Pricing** | $0.08 per image |
| **Type** | Image-only (generateImage) |

## Quick Start

```typescript
import { experimental_generateImage as generateImage } from 'ai';

const result = await generateImage({
  model: 'google/imagen-4.0-ultra-generate-001',
  prompt: 'Photorealistic portrait of a professional, studio lighting',
  aspectRatio: '3:4',
});

const image = result.images[0];
console.log(image.base64);
```
