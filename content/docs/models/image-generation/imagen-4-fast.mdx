# Imagen 4 Fast

Google's fast image generation model.

## Specifications

| Property | Value |
|----------|-------|
| **Model ID** | `google/imagen-4.0-fast-generate-001` |
| **Pricing** | $0.02 per image |
| **Type** | Image-only (generateImage) |

## Quick Start

```typescript
import { experimental_generateImage as generateImage } from 'ai';

const result = await generateImage({
  model: 'google/imagen-4.0-fast-generate-001',
  prompt: 'A futuristic city skyline at dusk',
  aspectRatio: '16:9',
});

const image = result.images[0];
console.log(image.base64);
```

## Features

### Fast Generation

Optimized for quick response times:

```typescript
const result = await generateImage({
  model: 'google/imagen-4.0-fast-generate-001',
  prompt: 'Modern office building exterior',
  aspectRatio: '16:9',
});
```

### Multiple Images

Generate multiple variations:

```typescript
const result = await generateImage({
  model: 'google/imagen-4.0-fast-generate-001',
  prompt: 'Abstract art with geometric shapes',
  n: 4, // Generate 4 images
  aspectRatio: '1:1',
});

console.log(`Generated ${result.images.length} images`);
for (const image of result.images) {
  // Process each variation
}
```
