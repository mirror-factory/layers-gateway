# Gemini 2.5 Flash Image

Multimodal model that generates both text and images in a single call.

## Specifications

| Property | Value |
|----------|-------|
| **Model ID** | `google/gemini-2.5-flash-image` |
| **Context Window** | 1M tokens |
| **Pricing** | Per token (text) + image |
| **Type** | Multimodal (generateText) |

## Quick Start

Unlike image-only models, use `generateText` with image modalities:

```typescript
import { generateText, createGateway } from 'ai';

const gateway = createGateway({
  apiKey: process.env.AI_GATEWAY_API_KEY
});

const result = await generateText({
  model: gateway('google/gemini-2.5-flash-image'),
  prompt: 'Generate an image of a sunset over mountains and describe it',
  providerOptions: {
    google: {
      responseModalities: ['TEXT', 'IMAGE'],
    },
  },
});

// Text description
console.log(result.text);

// Generated images
if (result.files) {
  for (const file of result.files) {
    if (file.mediaType?.startsWith('image/')) {
      console.log('Image:', file.uint8Array?.length, 'bytes');
    }
  }
}
```

## Features

### Combined Text and Image

Generate descriptive text alongside images:

```typescript
const result = await generateText({
  model: gateway('google/gemini-2.5-flash-image'),
  prompt: 'Create a logo for a coffee shop called "Morning Brew" and explain your design choices',
  providerOptions: {
    google: {
      responseModalities: ['TEXT', 'IMAGE'],
    },
  },
});

console.log('Design explanation:', result.text);
// result.files contains the generated logo
```

### Conversation with Images

```typescript
const result = await generateText({
  model: gateway('google/gemini-2.5-flash-image'),
  messages: [
    { role: 'user', content: 'Draw a simple house' },
    { role: 'assistant', content: 'Here is a simple house...' }, // Previous response
    { role: 'user', content: 'Now add a garden in front of it' },
  ],
  providerOptions: {
    google: {
      responseModalities: ['TEXT', 'IMAGE'],
    },
  },
});
```

### Saving Generated Images

```typescript
import fs from 'fs';

const result = await generateText({
  model: gateway('google/gemini-2.5-flash-image'),
  prompt: 'Generate an illustration of a robot',
  providerOptions: {
    google: {
      responseModalities: ['TEXT', 'IMAGE'],
    },
  },
});

if (result.files) {
  for (let i = 0; i < result.files.length; i++) {
    const file = result.files[i];
    if (file.uint8Array) {
      fs.writeFileSync(`image-${i}.png`, file.uint8Array);
    }
  }
}
```
