# FLUX 2 Klein 4B

Lightweight 4B parameter model for fast, efficient generation.

## Specifications

| Property | Value |
|----------|-------|
| **Model ID** | `bfl/flux-2-klein-4b` |
| **Parameters** | 4 billion |
| **Pricing** | $0.014 per megapixel |
| **Type** | Image-only (generateImage) |

## Quick Start

```typescript
import { experimental_generateImage as generateImage } from 'ai';

const result = await generateImage({
  model: 'bfl/flux-2-klein-4b',
  prompt: 'Modern office interior design',
  aspectRatio: '16:9',
});

const image = result.images[0];
console.log(image.base64);
```

## Features

### Fast Generation

Optimized for speed:

```typescript
// Generate multiple images quickly
const prompts = [
  'Modern kitchen design',
  'Cozy living room',
  'Minimalist bedroom',
  'Home office setup',
];

const images = await Promise.all(
  prompts.map(prompt =>
    generateImage({
      model: 'bfl/flux-2-klein-4b',
      prompt,
      aspectRatio: '16:9',
    })
  )
);
```
