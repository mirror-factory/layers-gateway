# Welcome to Layers

Layers is a platform for building human-AI coordination tools. It provides:

- **Unified Credit System** - Single balance across all AI operations
- **Multi-Provider Support** - 23 language models across 5 providers
- **Vercel AI Gateway** - Unified access to Anthropic, OpenAI, Google, Perplexity, and Morph

## Quick Start

```bash
# Install the SDK
bun add @layers/models @layers/credits

# Or with npm
npm install @layers/models @layers/credits
```

## Supported Providers

| Provider | Models | Capabilities |
|----------|--------|--------------|
| **Anthropic** | Claude 4.5 (Haiku, Sonnet, Opus) | Text, Vision, Tools, Thinking |
| **OpenAI** | GPT-4o, GPT-5, GPT-5.1 variants | Text, Vision, Tools, Codex |
| **Google** | Gemini 2.5/3 Flash/Pro/Image | Text, Vision, Multimodal, Image Gen |
| **Perplexity** | Sonar, Sonar Pro, Reasoning Pro | Web Search, Reasoning |
| **Morph** | V3 Fast, V3 Large | Fast text editing |

## Architecture

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   Your App  │ ──▶ │   Vercel    │ ──▶ │ AI Provider │
│             │     │  AI Gateway │     │             │
└─────────────┘     └─────────────┘     └─────────────┘
                          │
                          ▼
                    ┌─────────────┐
                    │   @layers   │
                    │   credits   │
                    └─────────────┘
```
